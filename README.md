## Instalacion
1. Clonar el repositorio:
```bash
git clone https://github.com/ignaciopadula/challengeQA.git
```
2. Desde la terminal del proyecto, instalar node 18
```bash
nvm install 18
```
3. Usar node 18
```bash
nvm use 18
```

4. Instalar dependencias
```bash
npm install
```
